Boas-vindas! Este site apresenta uma coleção de jogos desenvolvidos exclusivamente utilizando HTML, CSS e JavaScript. O principal objetivo do projeto é abster-se do uso de qualquer outra ferramenta, concentrando-se apenas nas três bases do desenvolvimento.

A intenção é explorar ao máximo todas as funcionalidades dessas linguagens e proporcionar uma comparação para compreender a utilidade e importância dos frameworks. O projeto também visa demonstrar a extensa quantidade de código necessária para reproduzir funções simples presentes nos frameworks.

Atualmente, disponibilizei apenas dois jogos: Flappy Bird e Genius

Além disso, outros jogos já estão em desenvolvimento

Funcionalidades, até o momento, de destaque em cada linguagem:

HTML - Buttons e sidebar.

CSS - Estilização avançada, animações, transições e responsividade.

Javascript - Lógica complexa dos jogos, gestão de pontuação e fim de jogo, assincronia, temporizadores e melhoria na responsividade.
